# URL Shortener boilerplate

## Getting started

Install dependencies:

```bash
npm i
```

Run the app:

```bash
npm run dev -w app
```

Run the api:

```bash
npm run dev -w api
```

Run the api tests:

```bash
npm run test -w api
```

The app and api restart on code change.

Format code:

```bash
npm run format
```
